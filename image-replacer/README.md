# 圖片替換工具

一個用於替換網站中所有圖片的Electron桌面應用程式。

## 功能特點

- 📸 **圖片選擇**：為每個圖片位置選擇對應的圖片文件
- 👁️ **即時預覽**：選擇圖片後立即在網頁預覽中查看效果
- 📄 **多頁面預覽**：可以切換查看不同頁面（作品集、服務、Visual、Tech）
- ✅ **批量替換**：確認後一次性替換所有選中的圖片
- 📊 **進度顯示**：顯示替換進度和結果

## 安裝與運行

### 1. 安裝依賴

```bash
cd image-replacer
npm install
```

### 2. 運行應用

```bash
npm start
```

### 開發模式（帶開發者工具）

```bash
npm run dev
```

## 使用說明

1. **選擇圖片**
   - 在左側列表中點擊「選擇圖片」按鈕
   - 從文件選擇對話框中選擇對應的圖片
   - 圖片會立即顯示在預覽區域

2. **預覽效果**
   - 右側會顯示當前頁面的預覽
   - 可以切換不同頁面查看效果
   - 選擇圖片後會自動更新預覽

3. **確認替換**
   - 選擇完所有需要的圖片後，點擊「確認並替換」按鈕
   - 確認對話框中會顯示將要替換的圖片數量
   - 點擊「確定替換」開始執行

4. **查看結果**
   - 替換完成後會顯示結果對話框
   - 顯示成功和失敗的圖片列表

## 圖片配置

所有需要替換的圖片位置都定義在 `image-config.json` 中，包括：
- 圖片ID和名稱
- 目標路徑
- 所屬HTML文件
- 分類

## 注意事項

- 請確保在運行應用前，網站文件（HTML）已經準備好
- 圖片會自動複製到對應的 `images/` 資料夾
- 如果目標資料夾不存在，會自動創建
- 建議在替換前備份原始文件

## 技術架構

- **Electron** - 桌面應用框架
- **Node.js** - 後端文件操作
- **HTML/CSS/JavaScript** - 前端界面
