# 圖片替換工具 - 使用說明

## 快速開始

### 1. 安裝依賴

在 `image-replacer` 資料夾中執行：

```bash
npm install
```

這會安裝 Electron 和所有必要的依賴。

### 2. 啟動應用

```bash
npm start
```

或開發模式（帶開發者工具）：

```bash
npm run dev
```

## 使用流程

### 步驟1：選擇圖片

1. 應用啟動後，左側會顯示所有需要替換的圖片列表
2. 點擊「選擇圖片」按鈕
3. 從文件選擇對話框中選擇對應的圖片文件
4. 圖片會立即顯示預覽縮圖

### 步驟2：預覽效果

1. 右側會顯示當前頁面的網頁預覽
2. 選擇圖片後，預覽會自動更新（如果可能）
3. 可以點擊頂部的頁面按鈕切換查看不同頁面：
   - 作品集
   - 服務
   - Visual
   - Tech

### 步驟3：確認並替換

1. 選擇完所有需要的圖片後，點擊右上角的「確認並替換」按鈕
2. 確認對話框會顯示將要替換的圖片數量
3. 點擊「確定替換」開始執行
4. 等待替換完成，查看結果

## 功能說明

### 圖片列表

- **分類篩選**：可以按分類篩選圖片（作品集、服務頁面、Hero區等）
- **搜尋功能**：可以搜尋圖片名稱或分類
- **狀態顯示**：綠色標籤表示已選擇圖片，灰色表示未選擇

### 預覽功能

- **即時更新**：選擇圖片後會嘗試在預覽中更新
- **多頁面切換**：可以切換查看不同HTML頁面的預覽效果
- **圖片預覽**：每個圖片項目都會顯示縮圖預覽

### 替換功能

- **批量處理**：一次可以替換多張圖片
- **自動創建資料夾**：如果目標資料夾不存在，會自動創建
- **進度顯示**：顯示替換進度和結果
- **錯誤處理**：如果某張圖片替換失敗，會顯示錯誤信息

## 注意事項

1. **備份建議**：在執行替換前，建議備份 `images/` 資料夾
2. **圖片格式**：支援 JPG、PNG、GIF、WebP 格式
3. **圖片路徑**：所有圖片會自動複製到對應的 `images/` 子資料夾
4. **預覽限制**：由於瀏覽器安全限制，某些情況下預覽可能無法即時更新，這是正常的

## 疑難排解

### 應用無法啟動

- 確保已執行 `npm install`
- 檢查 Node.js 版本（建議 16+）

### 預覽無法顯示

- 這是正常的，由於瀏覽器安全限制
- 可以手動刷新預覽或切換頁面
- 實際替換後的圖片會正常顯示

### 圖片替換失敗

- 檢查圖片文件是否存在
- 檢查目標資料夾權限
- 查看錯誤信息了解具體原因

## 技術支援

如有問題，請檢查：
1. 控制台錯誤信息（開發模式下按 F12）
2. 文件權限設置
3. 圖片文件格式是否正確
