## 圖片規格建議

- **格式**：JPG（推薦）或 PNG
- **作品集圖片**：建議尺寸 800x600px 或更大（4:3比例）
- **Hero區圖片**：建議尺寸 1200x1500px 或更大（4:5比例）
- **檔案大小**：單張圖片建議不超過 500KB
- **品質**：使用適當的壓縮以平衡品質與檔案大小

## 注意事項

1. 圖片檔案名稱必須與上述列表完全一致（區分大小寫）
2. 如果圖片不存在，瀏覽器會顯示破圖，請確保所有圖片都已正確放置
3. 建議使用圖片壓縮工具優化檔案大小，以提升網站載入速度
4. 所有圖片路徑已在HTML中設定完成，只需將對應的圖片檔案放入正確的資料夾即可
